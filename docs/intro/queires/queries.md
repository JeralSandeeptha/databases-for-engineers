# Database Queries

A `database query` is a formal request for data or an action sent to a database management system (DBMS).

It allows users, applications, and administrators to `retrieve`, `filter`, `calculate`, `modify`, or `delete` information stored within database tables

Think of it as asking a highly specific question or giving a precise command that the database translates and executes

## Query Types

<br/>

### Data Definition Language (DDL)

- To defined the data structures, designing and shaping the structure of a database.

### Data Query Language (DQL)

- Retrieving data from databases

### Data Control Language (DCL)

- To control access to data in a database.

### Data Manipulation Language (DML)

- Modifying or handling data within a database
